JS数组定义

数组对象是使用单独的变量名来存储一系列的值;就是用[ ]包起来的都是数组

1.var ary = [1,2,3,4,5];//这就是一个数组
那么数组有什么意义呢?
1.var num = 1;
2.var num2 = 2;
3.var num3 = 3;
4....
5.
6.如果我想找出某一个数字或者几个数字很好找,那么要是找出成百上千个数字呢.

上述最好的方法就是用数组;数组可以用一个变量名存储所有的值,并且可以用变量名访问任何一个值.
数组中的每一个元素都有自己的ID(索引值)一遍它可以很容易被访问到

索引值

上述说到数组的索引;那么就来说一说索引值

1.var num = [1,2,3,2,1];
2.//数组也是对象数据类型,对象数据类型都是由属性名:属性值组成;我们的数组也不例外
3. [0:1,1:2,2:3,3:2,4:1]
4. 上述冒号前面的就是数组中的属性值也叫做索引

索引都是从0开始的

数组属性

属性 描述
constructor 返回创建数组对象的原型函数
length 每个数组自带的长度属性,代表数组的个数
prototype 原型;利用这个属性可以向数组对象添加属性或方法

ary[ary.length-1];找到末尾项
ary[ary.length] = 5;向数组末尾增加一项(5)
ary[ary.length–];删除末尾最后一项

Array对象属性(数组原型上面的属性)

数组的方法分别可以分为增删改查

  1. [].pop:删除数组末尾最后一个元素;

    返回值:删除的元素

    var ary = [1,2,3,4,5,6];
    ary.pop(); //返回值:[6]
    console.log(ary);//输出[1,2,3,4,5]

  2. [].push:向数组的末尾添加一个或多个元素
    返回值:新数组的长度(length值)

    var ary = [1,2,3,4,5,6];
    ary.push(9);//返回值:7(新数组长度)
    console.log(ary);//输出[1,2,3,4,5,6,9]
  3. [].shift:删除数组的第一个元素;
    返回值:被删除的第一个元素

    var ary = [1,2,3,4,5,6];
    ary.shift(); //返回值:[1]
    console.log(ary);//输出[2,3,4,5,6]
  4. [].unshift:向数组开头增加一个或多个元素
    返回值:新数组的长度

    var ary = [1,2,3,4,5,6];
    ary.unshift(7); //返回值:[7]
    console.log(ary);//输出[7,1,2,3,4,5,6]

  5. [].splice:向数组中插入,删除或者替换元素(增删改)

    修改:

    返回值:被删除的内容

语法:ary.splice(x,y,z)

从索引x开始删除y个元素,用z替换(z可以是多个值)

var ary = [11,22,33]
ary.splice(0,2,3);//返回值:[11,22]
console.log(ary);//输出[3,33]

增加:
返回值:空数组

语法:ary.splice(x,0,z);

从索引x开始一个都不删除(0)把z增加到索引x前面,返回一个空数组;原数组不变

>

var ary = [11,22,33]
ary.splice(1,0,3);//返回值:[]
console.log(ary);//输出[11,3,22,33]

删除:

返回值:删除的内容以一个新数组返回

语法:ary.splice(x,y);

从索引x开始删除y个元素,包含y ;


var ary = [11,22,33,23,55];
ary.splice(1,2);//返回值:[22,33]
console.log(ary);//输出:[11,23,55]


ary.splice(x);
//从索引x开始,删除到末尾,把删除的内容以一个新数组返回


ary.splice(0);
//清空数组
//返回原来的数组


ary.splice();
//如果不传参数的话,返回一个空数组,原有数组不发生改变

  1. [].slice:实现数组的截取

    返回值:返回查找到的元素以新数组返回,原有数组不发生改变

    语法:ary.slice(x,y);

    从索引x项找到y项之前(不包含y项),把查找的内容以一个新数组返回;m必须大于n

    1.var ary = [11,22,33,23,55];
    2.ary.slice(1,3);//返回值:[22,33]
    3.console.log(ary);//输出:[11,22,33,23,55]
    1.ary.slice(x);
    2.//从索引x开始,一直找到数组的末尾,将找到的部分以一个新数组返回,原有数组不变

    1. [].concat:合并数组

    返回值:合并的新数组

1.var ary = [1,2];
2.var ary1 = [2,1];
3.ary.concat(ary1);
4.//输出:[1,2,2,1]
1.var ary = [1,2];

  1. [].join(''):按照每一个分隔符,把数组中的每一项拼接成一个字符串;原有数组不变
